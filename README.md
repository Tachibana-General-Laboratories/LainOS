# LainOS

[QEMU Raspberry Pi 3](https://github.com/bztsrc/qemu-raspi3)

## File Hierarchy

- `/bin/` contains executables
- `/etc/` configuration files
- `/sys/` virtual file system
- `/usr/` home directories
- `/var/` contains variable files
